char* aes_mode_error ( void  )

Returns the standardized error message for a wrong AES key mode.

Returns the standardized error message for when a wrong AES key mode was passed. Message is:

Fatal Error: an invalid aes mode was passed.
> Even though Rijndael supports several lengths of key bits, AES is defined to only support 128, 192, or 256 bits

A string for the specific error.

uint32_t rot_word ( uint32_t  inp)
inline

Applies the RotWord of the AES algorithm.

The function takes a word (which is 4 bytes) and performs a cyclic permutation. The permutation is as follows:

[a_0, a_1, a_2, a_3] -> [a_1, a_2, a_3, a_0]
Parameters
inpThe input word (32 bits or 4 bytes)
Returns
A word resulting from the permutation on the input word
